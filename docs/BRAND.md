# Zeravue Brand Guide (Working)

## Product Name

- Primary: `Zeravue`
- Korean display: `제라뷰`

## Product Structure

- Zeravue is the parent product.
- Night Sky is Theme 01.
- The legacy "Planetarium" naming now maps to the Night Sky theme experience.

## Promise

`Open a scene, breathe, stay for a while.`

## Tone

- calm
- immersive
- gentle
- emotionally readable

## Theme Naming Convention

- Use `Theme 01: Night Sky`, `Theme 02: Aurora Night`, etc.
- Keep controls and layout language product-wide where possible.
- Allow each theme to define its own mood and audio.
