# Zeravue Theme Expansion Notes

## Theme Strategy

All themes should live inside Zeravue and share a common viewer structure.

Each theme can have:

- its own atmosphere
- its own sound design
- its own movement language
- a small number of theme-specific controls

But the overall product interaction should remain familiar.

## Theme 1: Night Sky

Status: active primary theme

Focus:

- space viewing mode
- observer mode
- panorama mode
- constellation focus and tracking
- night-sky sketching
- fullscreen appreciation

Mood words:

- quiet
- suspended
- luminous
- contemplative

## Theme 2: Aurora Night

Concept:

The night sky remains, but aurora becomes the emotional centerpiece.

Possible features:

- moving aurora curtains
- aurora intensity control
- colder palette presets
- more atmospheric horizon glow

Why it is a good second theme:

- closest to the existing star theme
- can reuse much of the current sky system
- feels distinct without requiring a new product structure

## Theme 3: Underwater Panorama

Concept:

A slow, immersive underwater scene focused on drifting motion and depth.

Possible features:

- suspended particles
- slow fish silhouettes or light forms
- surface light shafts
- current intensity slider

Mood words:

- floating
- submerged
- slow
- weightless

## Theme 4: Rainforest in Rain

Concept:

A dense, rainy forest scene centered on rain, mist, and layered sound.

Possible features:

- rain intensity
- mist density
- leaf movement
- occasional distant thunder

Mood words:

- humid
- cocooned
- alive
- restful

## Theme 5: Cloud Panorama

Concept:

A high-altitude sky scene focused on bright clouds, horizon depth, and softness.

Possible features:

- cloud density
- height presets
- warm / cool daylight tones
- slow drift speed

Mood words:

- open
- airy
- bright
- gentle

## Shared Systems Across Themes

These should eventually become product-wide systems:

- scene selection
- fullscreen mode
- favorites
- saved viewing presets
- ambient sound controls
- slow camera choreography
- lightweight theme metadata

## Expansion Rule

Do not add a new theme until the night-sky theme is strong enough to serve as the reference implementation.

The first expansion target should be:

1. Aurora Night
2. Underwater Panorama
3. Rainforest in Rain
4. Cloud Panorama
